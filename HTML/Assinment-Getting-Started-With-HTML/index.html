4. Write a short note on Tag and element with an example?
In HTML, a tag is the text enclosed in angle brackets. For example,
      `<p>` (paragraph) is a tag. 

      An element refers to the entire unit, starting with the opening tag, encompassing the content, and concluding with the closing tag. 
      An example of an element is:
       `<p>This is a paragraph.</p>`. 
      
      Some elements, called "empty" or "void" elements, don't have content or closing tags, such as `<br>` (line break) or `<img>` (image).
      
      Tags are the building blocks of HTML, and elements provide structure and meaning to the content within a web page.
