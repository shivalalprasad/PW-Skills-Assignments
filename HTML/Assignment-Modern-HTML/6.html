HTML vs. HTML5

HTML HTML5
Uses div and span for layout and styling. Introduces semantic tags like <header>, <nav>, and <article> for better structure.
Doesn't support native audio and video elements. Includes <audio> and <video> tags for embedding multimedia content.
Requires JavaScript libraries (e.g., jQuery) for animations and interactivity. Offers the <canvas> element for dynamic graphics and animations using native JavaScript.
