List of HTML Tags

  1. <html> - The root element that defines the entire HTML document. The `lang` attribute specifies the language of the document.
  2. <head> - Contains meta-information about the HTML document, such as the title, character set, and links to external stylesheets or scripts.
  3. <title> - Defines the title of the HTML document, which is displayed in the browser's title bar or tab.
  4. <body> - Contains the visible content of the HTML document, including text, images, links, and other elements.
  5. <h1> - <h6>>- Heading elements that define six levels of section headings. <h1> is the highest (most important) level and <h6> is the lowest.
  6. <p> - Defines a paragraph of text.
  7. <a href="https://www.example.com">Link Text</a> - Creates a hyperlink to another webpage or resource. The `href` attribute specifies the URL of the link destination, and the text between the opening and closing tags is the visible link text.
  8. <img src="image.jpg" alt="Image Description"> - Embeds an image in the document. The `src` attribute specifies the image file path, and the `alt` attribute provides a text description for accessibility.
  9. <ul> - Defines an unordered list (a list with bullet points).
  10. <ol> - Defines an ordered list (a list with numbered items).
  11. <li> - Defines a list item within an unordered or ordered list.
  12. <div> - A generic container element that is often used for styling or scripting purposes.
  13. <span> - An inline container element that is often used for styling or scripting purposes.
